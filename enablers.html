About this website

This website was built with Foundation For Sites Framework, with custom SASS/CSS, YAML, JavaScript, HTML, and Markdown elements. It was entirely designed and coded by Nathan Tarter. The full project is up on GitHub as an open source flat-file website template.

Building this site virtually from scratch has been a labor of love, and just 6 months of relaxation time, hair-pulling, and procrastination. Hope you enjoy. If you want a super fast, simple and impactful website that you can update yourself, let me know. I have lots of friends who could help you with that mis-guided endeavor.
